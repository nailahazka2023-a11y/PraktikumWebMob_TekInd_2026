Penjelasan Penggunaan Kode

1. Setup Awal

document.getElementById("formProduksi")
.addEventListener("submit", function (event) {

    event.preventDefault();

Baris kode ini mengambil elemen form berdasarkan ID, kemudian menambahkan event listener untuk mendeteksi aksi submit. Fungsi event.preventDefault() digunakan agar halaman tidak melakukan refresh otomatis saat form dikirim.

2. Mengambil Data Input

let biayaBahanBaku = Number(document.getElementById("bahanBaku").value);
let biayaTenagaKerja = Number(document.getElementById("tenagaKerja").value);
let biayaOverhead = Number(document.getElementById("overhead").value);
let totalProduksi = Number(document.getElementById("totalProduksi").value);

Data yang dimasukkan pengguna melalui field HTML diambil menggunakan getElementById(), lalu dikonversi ke tipe Number supaya dapat diproses dalam operasi aritmatika.

3. Validasi Input

if (totalProduksi <= 0) {
    output.textContent = "Total produksi tidak valid.";
    return;
}

Bagian ini berfungsi untuk memeriksa apakah total produksi bernilai nol atau kurang dari nol. Jika kondisi tersebut terjadi, maka proses perhitungan dihentikan menggunakan return.

4. Perhitungan Biaya

let totalPerUnit =
(biayaBahanBaku + biayaTenagaKerja + biayaOverhead) / totalProduksi;

Total per Unit = (Bahan Baku + Tenaga Kerja + Overhead) ÷ Total Produksi

5. Logika If / Else

let status;

if (totalProduksi < 100) {
    status = "Biaya Tinggi (Ekonomi Skala Kecil)";
} else {
    status = "Biaya Efisien";
}

Struktur percabangan dimanfaatkan untuk menentukan kategori biaya berdasarkan jumlah unit produksi. Jika produksi kurang dari batas tertentu, maka dikategorikan sebagai biaya tinggi, sedangkan produksi dalam jumlah besar dianggap lebih efisien.

6. Menampilkan Output

let output = document.getElementById("hasilOutput");

output.innerHTML =
"Total Biaya per Unit: Rp " +
totalPerUnit.toLocaleString("id-ID", { minimumFractionDigits: 2 }) +
"<br>Status Produksi: " + status;

Nilai hasil perhitungan ditampilkan pada elemen dengan ID hasilOutput menggunakan properti innerHTML, sehingga informasi total biaya dan status produksi dapat ditampilkan secara dinamis di halaman.
