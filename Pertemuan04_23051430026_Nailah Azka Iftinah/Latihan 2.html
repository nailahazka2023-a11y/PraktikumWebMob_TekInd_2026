Program ini meminta pengguna memasukkan kode shift kerja dalam bentuk angka:

  • 1 = Pagi
  • 2 = Siang
  • 3 = Malam

Data tersebut diterima melalui fungsi prompt(), kemudian diproses menggunakan struktur percabangan if / else if / else. Hasil evaluasi ditampilkan pada console menggunakan console.log().

Step 1 – Mengambil Input dari Pengguna


let input = Number(
    prompt("Masukkan kode shift (1=Pagi, 2=Siang, 3=Malam)")
);

Fungsi prompt() berperan untuk menerima masukan dari pengguna melalui kotak dialog pada browser. Nilai yang dihasilkan oleh fungsi ini secara default bertipe String.

Agar dapat dilakukan perbandingan numerik, nilai tersebut dikonversi menggunakan fungsi Number() sehingga berubah menjadi tipe Number dan bisa dibandingkan dengan operator perbandingan ketat (===).

Jika pengguna memilih Cancel, maka nilai yang diterima adalah null. Apabila data yang dimasukkan bukan angka, maka hasil konversinya akan menjadi NaN.

Step 2 – Logika Percabangan If / Else If / Else


if (input === 1) {
    console.log("Shift Anda adalah Pagi");
} else if (input === 2) {
    console.log("Shift Anda adalah Siang");
} else if (input === 3) {
    console.log("Shift Anda adalah Malam");
} else {
    console.log("Shift Tidak Valid");
}

Struktur if / else if / else digunakan untuk melakukan pengecekan kondisi secara bertahap berdasarkan nilai yang dimasukkan. mengevaluasi kondisi secara berurutan.

Operator === dikenal sebagai strict comparison, yaitu operator yang membandingkan nilai sekaligus tipe datanya. Karena input telah diubah menjadi Number, proses evaluasi dapat dilakukan dengan lebih tepat.

Jika nilai yang diberikan tidak termasuk dalam kategori 1, 2, atau 3, sistem akan menampilkan pesan bahwa kode shift yang dimasukkan tidak valid.

Alur Logika Program

  1. Sistem meminta pengguna memasukkan kode shift kerja.
  2. Nilai input diubah menjadi tipe Number.
  3. Program melakukan pengecekan terhadap nilai 1, 2, dan 3.
  4. Jika sesuai, informasi shift akan ditampilkan.
  5. Jika tidak sesuai, muncul pesan bahwa input tidak valid.
Jalankan program dan periksa hasilnya melalui Console Browser dengan menekan F12.
