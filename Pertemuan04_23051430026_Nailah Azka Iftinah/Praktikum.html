Step 1 – Variabel dan Tipe Data


let namaMesin = "CNC-Mazak-01"; // String
let targetHarian = 500; // Number
let isOperational = true; // Boolean

console.log("Mesin: " + namaMesin);
console.log("Target: " + targetHarian);

Pada tahap ini dilakukan deklarasi variabel menggunakan kata kunci let. Variabel namaMesin bertipe string karena menyimpan data teks, sedangkan targetHarian bertipe number karena digunakan dalam proses perhitungan numerik. Output ditampilkan melalui console.log() untuk melihat hasil eksekusi program pada console browser.

Step 2 – Operator Aritmatika


let produksiPagi = 200;
let produksiSiang = 150;

let totalProduksi = produksiPagi + produksiSiang;
console.log("Total saat ini: " + totalProduksi);

let kekurangan = targetHarian - totalProduksi;
console.log("Kekurangan target: " + kekurangan);

Operator aritmatika penjumlahan (+) digunakan untuk menghitung total produksi dari dua shift kerja. Selanjutnya operator pengurangan (-) digunakan untuk mengetahui selisih antara target dan realisasi produksi. Proses ini menunjukkan penerapan dasar operasi matematika dalam JavaScript.

Step 3 – Percabangan Target Produksi


let jamOperasional = 1250; // dalam jam
let batasMaksimal = 1200; // batas sebelum maintenance wajib
console.log("--- Cek Status Maintenance ---");

if (jamOperasional >= batasMaksimal) {
 console.log("PERINGATAN: Mesin mencapai batas maksimal!");
 console.log("Status: MAINTENANCE WAJIB (Stop Produksi)");
} else if (jamOperasional > 1000) {
 console.log("Status: SIAP HATI-HATI (Segera jadwalkan maintenance)");
} else {
 console.log("Status: BERJALAN NORMAL");
}

Struktur percabangan if-else digunakan untuk menentukan apakah total produksi telah memenuhi target harian. Jika nilai totalProduksi lebih besar atau sama dengan targetHarian, maka sistem menampilkan pesan bahwa target tercapai.

Step 4 – Perhitungan Availability


let jamKerjaPlanned = 8; // Jam
let jamKerjaAktual = 6.5; // Jam (Ada 1.5 jam breakdown)

let availability = (jamKerjaAktual / jamKerjaPlanned) * 100;
availability = availability.toFixed(2);

console.log("Planned Time: " + jamKerjaPlanned + " Jam");
console.log("Actual Time: " + jamKerjaAktual + " Jam");
console.log("Availability: " + availability + "%");

if (availability >= 90) {
 console.log("Kategori: WORLD CLASS");
} else if (availability >= 80) {
 console.log("Kategori: BAIK (Tetap monitor)");
} else {
 console.log("Kategori: BURUK (Perlu investigasi penyebab breakdown)");
}

Availability dihitung berdasarkan rasio waktu kerja aktual terhadap waktu kerja rencana yang dikalikan 100%. Fungsi toFixed(2) digunakan untuk membatasi dua angka di belakang koma. Nilai tersebut kemudian dikategorikan menggunakan percabangan.

Pengelompokan kategori dilakukan dengan struktur if-else if untuk menentukan tingkat kinerja mesin, seperti WORLD CLASS, BAIK, atau BURUK berdasarkan standar tertentu.

Step 5 – Input Pengguna (prompt dan alert)


let namaOperator = prompt("Masukkan Nama Operator:");
let shiftKerja = prompt("Masukkan Shift (Pagi/Siang/Malam):");

if (shiftKerja === "Malam") {
 alert("Halo " + namaOperator + ", Shift malam memiliki tambahan uang makan sebesar Rp 
20.000.");

} else {
 alert("Halo " + namaOperator + ", Selamat bekerja. Tetap semangat!");
}

Fungsi prompt() digunakan untuk menerima input dari pengguna dan menyimpannya dalam variabel. Nilai yang dikembalikan selalu bertipe string, kecuali jika pengguna menekan Cancel maka nilainya menjadi null.

Struktur percabangan digunakan untuk mengevaluasi isi variabel shiftKerja. Jika bernilai "Malam", maka sistem akan memberikan notifikasi tambahan uang makan menggunakan fungsi alert().

Mekanisme ini menunjukkan interaksi sederhana antara pengguna dan sistem berbasis web, di mana input diproses dan menghasilkan output sesuai kondisi yang telah ditentukan.
